User Management API

This API provides endpoints for managing users using Express and Prisma.

Endpoints

Route Method Description
/users GET Retrieves a list of users (with filtering).
/users/:id GET Retrieves a single user by ID.
/users POST Creates a new user.
/users/:id PUT Updates an existing user.
/users/:id DELETE Deletes a user.
/ GET Welcome route.
GET /users

Retrieves a list of users. Supports filtering by name, email, or phone number.

Parameters (Query):
  • name (string, optional): Filters users by name (case-insensitive, partial match).
  • email (string, optional): Filters users by email (case-insensitive, partial match).
  • phoneNumber (string, optional): Filters users by phone number (partial match).
200 OK

Returns an array of user objects (JSON):

[
  {
    "id": "cll4mwkfz0000000000000000",
    "name": "John Doe",
    "email": "[email address removed]",
    "phoneNumber": "123-456-7890"
  },
  // ...
]
500 Internal Server Error

Error retrieving users:

{
  "error": "Failed to get users"
}
